# Changelog - @asyncflowstate/next

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-11

### Added

- Initial release of AsyncFlowState Next.js integration
- `useServerActionFlow` hook for Server Actions integration
- `useTransitionFlow` hook for React Transitions support
- Full Server Components compatibility
- Edge Function support
- App Router compatibility
- Pages Router support (legacy)
- TypeScript support with full type safety
- Streaming Response support
- Form integration with Server Actions

### Features

- Seamless Server Actions integration
- Automatic request deduplication
- Error boundary support
- Pending state management
- Optimistic updates for Server Actions
- Type-safe server function calls
- Full Next.js 13+ support
- Middleware compatibility
- API Routes integration
- Image optimization support

## [Unreleased]

### Planned Features

- Dynamic route segment caching
- Middleware hooks
